Typography

Oliver Reichenstein of Information Architecht wrote about how “The web is 95% typography” back in 2006 and it's even more true today, which is why Yggdrasil gives you complete control to typography with brilliant defaults. Every size and leading (line-height) matters.

Flexible & Fluid Grid

Smarter, more semantic markup that doesn't change, even when the underlying grid does. With fractional naming, everything makes more sense. And unlike other frameworks, Yggdrasil avoids using HTML5 elements like section when it's not appropriate.

!Normalize

Yggdrasil takes all your default typography, margin, padding, etc. and seamlessly makes it a part of normalizing. No more needless cascade overrides. Normalize.css (v3.0.1) and brings into the makes browsers render all elements more consistently and in line with modern standards. It precisely targets only the styles that need normalizing.

Nicolas Gallagher and Jonathan Neal

Preserves useful defaults

Natural box layout

Yggdrasil changes the box-sizing to border-box for all elements which applies a natural box layout model. It might be a small thing but it makes a world a difference, as recommended by Paul Irish.

Sass

Yggdrasil is built from the ground up with pre-processing using Sass, letting you use features like variables, nesting, mixins, inheritance and other nifty goodies that make writing CSS fun again.

Accessible (a11y)

The web shouldn't be limited to the few but available to everyone. Yggdrasil puts usability, accessibility and performance optimisation in the front seat and not as an afterthought.

Pattern library

Nenver struggle to mark up a form again, Yggdrasil comes with an assortment of the most common markup patterns, including Media components, Hero units, Tooltips and Carousels.

Prismatic & Polarize

Yggdrasil and its Prismatic will generate an appropriate color scheme based on a single value. Coupled with Polarize, your texts will always have sufficient contrast.

Everything is generated from $primaryColor, which you set in your _theme file.

Choose between Analogous, Triadic or Split color models and Yggdrasil will take care of the rest, including contrast for text colors (from perceived brightness instead of lightness).

Are you tired of memorizing obscure column names that hold little semantic value? Look no further, Yggdrasil is the salvation you've been looking for.

Instead of confusing names, Yggdrasil uses a fractional naming scheme for its more common columns, i.e. when you want a column that spans half the width, you simply give it a class of .span-half.

Always using the simplest fraction, i.e. there's no .two-quarter because that would be a half.
